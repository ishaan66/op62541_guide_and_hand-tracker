Repostories with code needed to set up the yocto build with TSN demos:

https://github.com/intel/iotg-yocto-bsp-public/tree/e3900/preempt-rt
https://github.com/intel/iotg_tsn_ref_sw

Files in repository:
1. TSN-BKC.pdf and TSN_UG_index.htm are the pdf and html form of the guide on hwo to run the 3 TSN demos on TSN capable boards with the yocto build installed
2. atom-soc-for-yocto-getting-started-guide.pdf is a guide on how to set up the yocto build on a linux machine running Ubuntu 14.04 to craete a USB from which yocto can be booted
3. 611026rev1-0-apl-i-preempt-rt-tsn-releasenotes.pdf contains notes on the TSN specifications demonstrated by the 3 demos
