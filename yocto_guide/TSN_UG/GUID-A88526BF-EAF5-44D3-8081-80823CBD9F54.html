Get Started

This section helps users get started using the Time Sensitive Networking (TSN) Reference Software for Linux. It includes the following subsections:

Build the Yocto Project*-Based Image on the Host Machine Describes installing the Yocto Project*-based BSP image, and adding Time Sensitive Networking Reference Software to the BSP. Once the image is built, users can follow the steps to create a bootable USB flash drive from the image to start the CRB.
Yocto Project* Commands Reference helpful commands for Yocto Project users that will be used in setting up the environment.
CPU Clock Optimization Provides steps to minimize packet drop due to CPU speed fluctuation. This can occur if:
  • Power saving mode is enabled OR
  • Intel SpeedStep® technology is not disabled
