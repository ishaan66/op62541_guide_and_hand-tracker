Build the Yocto Project*-Based Image on the Host Machine

This section describes installing the Yocto Project-based BSP image, and adding Time Sensitive Networking Reference Software to the BSP. Once the image is built, users can follow the steps to create a bootable USB flash drive from the image to start the CRB.

Description Estimated Time Taken
Step 1: Prepare the Host Machine and Build the BSP 1 hour (set up) + 8 hours (build)
Step 2: Create a Bootable USB Flash Drive 20-60 minutes
Step 3: Install Yocto Project* on Boards 15 mins (set up) +30 mins (install)

Note

Time estimates can vary widely depending on your network speed and system processing power. The estimates are created using an Intel® Core™ i7 processor.
